## 집필원칙

- Rmarkdown (`*.Rmd`) 형식으로 원고를 작성한다.
- 마이크로소프트 워드를 사용할 수도 있으나 pandoc을 사용하여 변환하는 과정을 거쳐야 하므로 피치 못할 경우에 사용한다.
- 모든 그림 파일은 pdf 형태 혹은 png 형태로 작성한다.
- R 패키지에 대해 기본적인 이해가 필요합니다. `rmarkdown`, `bookdown`, `knitr`, `kableExtra`
